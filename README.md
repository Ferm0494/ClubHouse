## ClubHouse

> The purpose of this project was to learn about authentication using devise gem for auths

## Built With

- Ruby
- Ruby on Rails

## Getting Started

You will need:
- Ruby
- Ruby on Rails
- Bundler

### Install

Clone the repo

`https://github.com/Ferm0494/forms`

Install dependencies

`bundle install`

Migrate the DB

`rails db:migrate`

Run the server

`rails s or and go into localhost:3000`
